# 🎓 ММТ 2026 - Барномаи Омодагӣ

Ин барнома барои кӯмак ба довталабони Ҷумҳурии Тоҷикистон сохта шудааст. Он маълумоти пурра дар бораи донишгоҳҳо, коллеҷҳо ва кластерҳоро пешниҳод мекунад.

## 🚀 Имкониятҳо
* **Рӯйхати Донишгоҳҳо ва Коллеҷҳо:** Маълумот дар бораи муассисаҳои таълимӣ.
* **Кластерҳо:** Рӯйхати ихтисосҳо мувофиқи кластерҳои 1, 2, 3, 4 ва 5.
* **Ҳисоби Бал:** Калкулятори махсус барои ҳисоб кардани балли шкалавӣ.
* **Тестҳо ва Ҷавобҳо:** Саволҳои машқӣ барои омодагӣ ба имтиҳонҳо.

---

## 🛠 Технологияҳо
* **Забон:** Python.
* **Китобхона:** Kivy / KivyMD (барои интерфейси графикӣ).

---

## 👤 Админ
Барои ворид шудан ба бахши танзимот, дар болои матни **"MMT 2026"** пахш кунед ва паролро ворид кунед.

---

## 📦 Тарзи истифода
1. Кодро ба **Pydroid 3** нусхабардорӣ кунед.
2. Файли `logo.png`-ро дар ҳамон папка ҷойгир кунед.
3. Тугмаи **Run**-ро пахш кунед.
